# AlphaWeights 打包与运行说明

## 1. 环境准备 (开发者)

确保您的开发环境中已经安装了 Python 和项目的依赖。

```bash
pip install -r requirements.txt
```

## 2. 执行打包

双击运行项目根目录下的 `package.bat` 脚本。

脚本会自动：
1. 安装 `pyinstaller` (如果未安装)。
2. 清理旧的构建文件。
3. 执行打包命令，将项目打包为单个 `.exe` 文件。

打包完成后，生成的可执行文件位于 `dist/AlphaWeights.exe`。

## 3. 发布与运行 (用户)

### 3.1 发布
您只需要将 `dist/AlphaWeights.exe` 文件发送给用户即可。

### 3.2 运行
用户在 Windows 上直接双击 `AlphaWeights.exe` 即可运行。

- **首次运行**：程序会在 exe 同级目录下自动创建 `data` 文件夹和 `fund_monitor.db` 数据库文件。
- **访问应用**：程序启动后会显示类似 `Running on http://127.0.0.1:5000` 的提示，用户在浏览器访问该地址即可。
- **关闭程序**：直接关闭黑色控制台窗口即可。

## 4. 注意事项

- **数据备份**：如果用户需要迁移程序，请务必同时复制 `data` 目录，否则数据会丢失。
- **端口冲突**：默认使用 5000 端口，如果用户电脑上该端口被占用，程序可能无法启动。
- **杀毒软件**：部分杀毒软件可能会对未签名的 exe 进行误报，添加信任即可。
